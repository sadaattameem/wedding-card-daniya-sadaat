Background music — I Found Love (Ed Sheeran)
============================================

Place your legally purchased MP3 here:

  music/i-found-love.mp3

How to add the song:
  1. Buy or download the track from Apple Music, Amazon, or another service you own.
  2. Export/save it as an MP3 file.
  3. Rename it to:  i-found-love.mp3
  4. Drop it in this folder (music/).

The site will:
  - Start playing softly when the guest taps the wax seal
  - Gradually swell the volume as the envelope opens and curtains part
  - Loop continuously (toggle with the button bottom-left)

To use a different filename, edit musicFile in script.js (WEDDING object).

Note: Do not commit copyrighted audio to a public GitHub repo unless you have
the rights to distribute it. This folder is gitignored for *.mp3 files.
